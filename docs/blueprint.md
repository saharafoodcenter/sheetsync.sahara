# **App Name**: SheetSync

## Core Features:

- Item Entry: Allow users to manually input items with names and expiry dates.
- Barcode Scanning: Add item details by scanning the barcode of the product and look up the product name from the database sheet.
- Google Sheets Integration: Sync item data (name, expiry date, batch) with a specified Google Sheet ('Inventory' sheet) for centralized storage and management.
- Expiry Notifications: Send notifications to users when items are nearing or have passed their expiry dates.
- Inventory Overview: Provide a dashboard displaying all entered items with expiry dates and visual cues for urgency.

## Style Guidelines:

- Primary color: Fresh Green (#90EE90) to evoke a sense of freshness and safety.
- Background color: Light Beige (#F5F5DC) for a clean and calming backdrop.
- Accent color: Warning Orange (#FFA500) to highlight nearing expiry dates.
- Body and headline font: 'PT Sans' for clear readability and a modern feel.
- Use simple, intuitive icons for item categories and actions.
- A clean, card-based layout to easily manage items.
- Subtle animations for expiry alerts.
